# 🐧 Linux - Basic Commands

## 📂 Navigation
- `pwd`: Show current path
- `cd folder`: Change directory
- `ls`: List files and folders
- `mkdir folder`: Create a folder
- `touch file`: Create a file

## 🧹 File Management
- `cp source dest`: Copy
- `mv source dest`: Move/Rename
- `rm file`: Delete
- `nano`, `vim`: Edit files
